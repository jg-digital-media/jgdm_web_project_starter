Web Starter Content

This is my web starter package file. You can use it if you wish to kick start your own web projects. It contains a number of NPM packages to get you started, including Angular, Vue, jQuery and React.js.

  • Angular
  • Express.js
  • Greensock (gsap)
  • jQuery
  • jsDOM
  • Phaser.js
  • React.js
  • Slick Carousel by Ken Wheeler
  • Bue.js
  • WebPack
As of 16-02-2022

Or you can simply expand on the CSS and HTML file provided. You're set up to use SASS if you wish but you can also just use a standard CSS file.
